# OnRide

**OnRide** is a ride-hailing Flutter app similar to Uber or Careem. It allows users to book rides, track drivers in real-time, and manage their ride history. The app focuses on smooth user experience, real-time updates, and secure operations.

---

## Features

- User registration and login  
- Book rides with pickup and drop locations  
- Real-time driver tracking on map  
- Ride history and receipts  
- Push notifications for ride status updates  
- Secure authentication and role-based access (user/driver)  
- Built with **Flutter** frontend and **Firebase / Node.js / FastAPI** backend
